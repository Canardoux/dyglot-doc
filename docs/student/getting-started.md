# Getting Started

Welcome to Dyglot Student.

Dyglot helps you learn using Courses created by teachers.
You don’t need to configure anything to begin.

Dyglot is designed so that you can start learning without understanding how Dyglot works.

## What happens the first time you open Dyglot

When you open Dyglot for the first time:

1. A default Course is already available.
2. A default View is selected.
3. You can start learning immediately.

## Start learning

On the home screen:

1. Choose a View.
2. (Optional) Choose a Filter.
3. Press **Practice**.

## During a session

- Cards are shown one by one.
- You answer using simple buttons.
- Your progress is saved automatically.

## Don’t worry

- You can stop at any time.
- Your progress is saved.
- You can reset a session later if needed.

# Getting Started

Welcome to Dyglot Student.

Dyglot lets you learn using Courses created by teachers.
You don’t need to configure anything to begin.

---

## First launch

When you open Dyglot for the first time:

- At least one Course is already available.
- A default View is selected.
- You can start learning immediately.

No account is required.
No setup is required.

---

## Start learning

From the home screen:

1. Select a **View**.
2. (Optional) Select a **Filter**.
3. Press **Practice**.

That’s all.

---

## During a session

- Cards are presented one by one.
- You answer using simple buttons.
- Your progress is saved automatically.

You can stop at any time and continue later.

---

## Your progress

Dyglot keeps track of your learning:

- Each View and Filter combination has its own Session.
- Sessions remember which cards you have already seen.
- You can reset a Session if you want to start over.

---

## Don’t worry

- You cannot lose your data by mistake.
- You can always reset and try again.
- Advanced features are optional.

Dyglot is designed so that you can start learning  
without understanding how Dyglot works.
